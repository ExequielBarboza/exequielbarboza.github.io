HTML

Hyper Text Markup Languaje
  • Es un lenguaje de marcado esencial para el desarrollo web porque nos permite definir la estructura de la página web, el contenido que se va a mostrar en el navegador.
  • Es el lenguaje estándar para crear paginas webs.
  • Le indica al explorador que y como mostrarlo.
  • Consiste de etiquetas.
  • Las etiquetas indican párrafos, títulos, vínculos, etc.
  • HTML trabaja en conjunto con [[CSS]] para crear las páginas web que usamos todos los días en el navegador. Algunas páginas web interactivas también usan [[JavaScript]].
  •  Los archivos HTML tienen extensión .html
